## 🔍 使用方法

### 方法1：SEMrush批量查询

**步骤1：登录SEMrush**
```
访问：https://www.semrush.com/
登录你的账号
```

**步骤2：进入Keyword Magic Tool**
```
1. 点击左侧菜单 "Keyword Research" → "Keyword Magic Tool"
2. 或直接访问：https://www.semrush.com/analytics/keywordmagic/
```

**步骤3：批量导入关键词**
```
1. 打开对应的txt文件（如 vpn-keywords.txt）
2. 复制所有关键词
3. 粘贴到SEMrush的搜索框
4. 点击 "Bulk Import" 或批量分析
```

**步骤4：导出数据**
```
1. 查看搜索量、难度、CPC等数据
2. 筛选适合你的关键词（建议：难度<40，搜索量>100）
3. 导出为Excel或CSV
```

---

### 方法2：Ahrefs批量查询

**步骤1：登录Ahrefs**
```
访问：https://ahrefs.com/
登录你的账号
```

**步骤2：进入Keywords Explorer**
```
1. 点击顶部菜单 "Keywords Explorer"
2. 选择搜索引擎（建议：Google China）
```

**步骤3：批量导入**
```
1. 打开txt文件，复制关键词
2. 粘贴到搜索框（每次最多1000个）
3. 点击搜索
```

**步骤4：分析数据**
```
查看指标：
- KD (Keyword Difficulty)：关键词难度
- Volume：月搜索量
- CPC：每次点击成本
- Traffic Potential：流量潜力
```

### 方法3：5118查询（中文SEO推荐）

**步骤1：访问5118**

```
1. 进入 "关键词工具" → "关键词挖掘"

```

## 📊 关键词筛选标准

### 优先级P0（必做）

```
✅ 搜索量：1000 - 10000/月
✅ 难度：<30
✅ 商业意图：高
✅ 示例：
   - 2024免费VPN节点
   - ChatGPT使用教程
   - 免费AI工具推荐
```

### 优先级P1（推荐）

```
✅ 搜索量：500 - 5000/月
✅ 难度：30-50
✅ 商业意图：中
✅ 示例：
   - V2Ray配置教程
   - AI绘画提示词
   - 白嫖VPS教程
```

### 优先级P2（备选）

```
✅ 搜索量：100 - 1000/月
✅ 难度：<60
✅ 商业意图：低-中
✅ 示例：
   - Clash规则配置
   - ComfyUI安装
   - 夸克网盘资源
```

## 📝 文章撰写流程

### 1. 选择关键词

```bash
# 从SEMrush导出的数据中选择：
1. 搜索量适中（500-5000）
2. 竞争度较低（KD<40）
3. 与博客主题相关
4. 有商业价值或用户需求

# 示例：
主关键词：2024免费VPN节点
长尾词：
  - 免费V2Ray节点分享
  - 每日更新免费节点
  - 高速免费代理
```

### 2. 创建文章模板

```markdown
---
title: 【2024最新】免费VPN节点 - 每日更新 | 主关键词
description: 包含主关键词和2-3个长尾词的自然描述，120-160字

keywords:
  - 主关键词（搜索量最高）
  - 长尾词1
  - 长尾词2
  - 长尾词3
  - LSI相关词1
  - LSI相关词2

date: 2024-11-17
author: 杰哥
category: VPN教程
tags:
  - VPN
  - 免费节点
  - V2Ray

cover: /images/cover.jpg
---
```

### 3. 内容优化

```
✅ 在标题中包含主关键词
✅ 前100字出现主关键词
✅ 每300字自然融入1次关键词
✅ 使用H2/H3标题包含长尾词
✅ 添加FAQ部分（回答"如何"、"什么是"）
✅ 文章末尾总结重复关键词
```

## 🎯 关键词使用技巧

### 1. 关键词密度

```
建议密度：1-2%
主关键词出现位置：
  ✅ H1标题（1次）
  ✅ 第一段前100字（1次）
  ✅ H2标题（2-3次）
  ✅ 正文（自然出现3-5次）
  ✅ 结论（1次）
```

### 2. 长尾词分布

```
文章结构：
  H1: 主关键词
  H2: 长尾词1
  H2: 长尾词2
  H2: 长尾词3
  H3: LSI相关词
```

### 3. 避免堆砌

```
❌ 错误：免费VPN节点、2024免费VPN节点、最新免费VPN节点...
✅ 正确：本文为您推荐2024年最新的免费VPN节点，包括V2Ray...
```

## 💡 实用建议

### 1. 关键词研究周期

```
每月更新：
  ✅ 查看新热门关键词
  ✅ 分析竞争对手变化
  ✅ 调整内容策略
  ✅ 优化低效关键词
```

### 2. 内容更新策略

```
优先更新：
  ✅ 排名在11-20位的文章（提升到前10）
  ✅ 有流量但跳出率高的文章
  ✅ 季节性/时效性关键词
```

### 3. 关键词组合技巧

```
组合公式：
  核心词 + 修饰词 + 年份
  
示例：
  免费VPN + 节点 + 2024
  ChatGPT + 教程 + 零基础
  AI工具 + 推荐 + 最新
```
